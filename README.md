# Impact of Social Media on Health
This Power BI project explores how social media usage affects health, sleep, mental well-being, and academic performance using a structured student-level dataset.
​

# Project Overview
- The dashboard analyzes records with fields including age, gender, academic level, country, average daily usage hours, most used platform, sleep hours per night, mental health score, academic impact, and overall impact.
​- It is designed to uncover patterns between social media behavior and lifestyle outcomes through interactive visuals and summary cards.
​

# Dashboard Highlights
- Most used social media platforms across the dataset.
​- Daily usage hours by country and demographic groups.
​- Sleep pattern analysis linked to social media usage.
​- Mental health and overall impact distribution as positive, neutral, or negative.
​- Academic performance effects based on usage behavior.
​
# Tools Used
- Power BI for dashboard creation and interactive visualizations.
- CSV dataset for data modeling and analysis.
​- Data cleaning and transformation in Power Query.

# Key Insights
- Higher daily social media usage often appears alongside lower sleep hours and more negative overall impact labels in the dataset.
​- The dataset also enables comparison across countries, genders, academic levels, and platforms to identify behavior trends.
​
